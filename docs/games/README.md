# Games catalogue

Design docs for each game built on the engine, one folder per game. See the root
[`CONTEXT-MAP.md`](../../CONTEXT-MAP.md) for how these contexts relate.

Each game folder holds:

- `CONTEXT.md` — the game's glossary (canonical vocabulary).
- `grill.md` — the grilling doc: design record + design→engine mapping.
- `adr/NNNN-*.md` — decisions specific to that game.
