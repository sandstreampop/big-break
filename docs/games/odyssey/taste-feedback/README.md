# Taste feedback — verdict archive

The Odyssey runs the same taste loop as love-island (grill Q16): batches of
authored copy go to the human for verdicts (`love`/`like`/`dislike`), archived
here in the LI JSON shape
(`docs/games/love-island/taste-feedback/verdicts-*.json` is the template).
Recurring dislikes become blocklist entries in [`taste.mjs`](../taste.mjs) +
[`VOICE.md`](../VOICE.md); ⭐ exemplars are promoted into
[`GUIDING_EXAMPLES.md`](../GUIDING_EXAMPLES.md).

**Batch zero is pending:** the canon samples in `GUIDING_EXAMPLES.md` were
authored fresh under the laws (the grilling transcript's approved samples were
not checked into the repo) and await human verdicts before the deck is written
against them.
